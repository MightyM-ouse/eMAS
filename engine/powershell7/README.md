# eMAS PowerShell 7 Adapter Boundary

This folder contains technical adapters for PowerShell 7.6 LTS.

Use this adapter for Pre-Migration Readiness and Post-Migration Verification on Windows. macOS PowerShell 7.6 is permitted for development and pure unit/fixture testing only.

Adapter code may use PowerShell 7.6 technical capabilities such as controlled parallel processing or improved UTF-8 behavior. It must not change business results compared with the shared core contract.
