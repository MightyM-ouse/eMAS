# Shared PowerShell Engine

This folder contains reusable technical processing modules used by all three eMAS phases.

Planned modules:

- configuration;
- discovery;
- classification;
- validation;
- effort;
- readiness;
- reconciliation;
- reporting and OpenXML;
- logging;
- common utilities.

The engine loads the reviewed runtime JSON. It must not read the internal XLSM mapping workbook or generate the runtime JSON.
