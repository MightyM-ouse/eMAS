# Historical eMAS Version 2 Documentation Pack

**Status:** Superseded

The Version 2 Word pack is retained outside this public repository as historical evidence. It is not an implementation authority.

The current successor baseline is listed in `docs/CANONICAL_DOCUMENT_INDEX.md`, led by Enterprise Requirements v3.1 and the approved configuration specifications.

Historical concepts that must not be reused include PowerShell reading the XLSM, PowerShell-generated JSON, editable IsActive lifecycle control, flat DossierType, NotAssessed as RAG, combined findings/recommendations and flat rules.

The internal archive must retain original files, checksums, review evidence and this supersession notice.
