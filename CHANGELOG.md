# Changelog

All material changes to eMAS source, configuration contracts, architecture, skills, templates and controlled documentation are recorded here.

## Unreleased

### Added

- canonical repository and governance structure;
- Enterprise Requirements v3.1 and Effective configuration baselines;
- frozen Normalized Relationship Matrix v1.0 and Logical Data Dictionary v1.0;
- Effective Runtime JSON Schema 1.0.0, modular definitions and synthetic fixture suite;
- independent schema/semantic validator, tests and CI workflow;
- Effective Solution Architecture v1.0;
- Effective Pre-Sales, Pre-Migration and Post-Migration phase contracts v1.0;
- seven Effective operational LLM skills for configuration, schema, PowerShell, regulatory content, reporting, review and defect investigation;
- machine-readable operational skill catalogue;
- independent operational-skill validator, unit tests and focused CI workflow.

### Changed

- Project Flow promoted to v2.0 and synchronized with configuration release validation, common startup, phase flows and failure paths;
- Repository Architecture promoted to v1.1 and synchronized with schema fixtures, phase contracts and package boundaries;
- canonical, documentation and LLM context indexes now route implementation through Effective operational skills;
- skill template now requires DecisionReferences, task-specific canonical sources and testable completion evidence;
- phase guidance includes controlled template/report boundaries, baseline behavior and the `MigrationSummary.xlsx` interface;
- script, engine, UI and template guidance references Effective architecture contracts;
- Pre-Sales excludes readiness terminology, WPF and raw-score disclosure by default;
- Pre-Migration baseline requirements include stable identifiers, scope, exclusions, limitations and integrity evidence;
- Post-Migration requires the approved baseline and preserves discrepancy/accepted-difference traceability;
- reporting rules keep EvaluationStatus, RAG, ValueSource, Confidence and ReviewRequired distinct.
